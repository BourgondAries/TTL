\subsection{Detailed Description}
A thread waiting for a task. 

This class is mainly used internally by the threadpool. This \hyperlink{classttl_1_1_worker}{Worker} thread is easily extended, as one could pass any function into issue\-Work. For example; in Thread\-Pool this class is used, and a function that encapsulates both thread management and the actual work itself will be sent into the \hyperlink{classttl_1_1_worker}{Worker} from within Thread\-Pool.
