Detailed Description

A benchmarking class.

A class designed for handling benchmarks of functions. It is possible to use member functions, lambdas, function pointers and const function pointers (normal functions).

void ttl::Benchmark::run ( Args &&...  args)
inline

Actual running algorithm.

Will loop over the function n times specified by setIterations or by the constructor. The running time will be added to the old running time and then halfed in order to approximate an average.
