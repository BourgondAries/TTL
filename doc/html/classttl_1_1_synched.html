Detailed Description

template<typename T>
class ttl::Synched< T >

Encapsulator that makes objects thread safe.

Allows multiple reads, but only single writes. This implementation is ALWAYS thread-safe, unless multiple readers edit mutable variables simultaneously. This depends on the class at hand. If your class has mutable fields, then make sure to make all const functions take note of that (and protect it). Another option is to make the mutable a Synched<> object.
